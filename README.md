## Smart Pizza POS & Restaurant Management System

An offline desktop Point of Sale (POS) and Restaurant Management System built with **Electron.js**, **Node.js**, **Express.js**, and **MySQL**.

This project is designed as a commercial-grade desktop application for pizza shops and restaurants to streamline billing, inventory management, customer management, reporting, and kitchen operations.

---

## 📖 Project Description

Smart Pizza POS is a desktop-based restaurant management system that helps restaurant owners efficiently manage their daily operations without relying on cloud-based subscription software.

The application is designed to work completely offline while providing fast billing, inventory automation, customer management, business analytics, and thermal receipt printing.

## ⚙️ Installation

### Prerequisites

- Node.js
- MySQL Server
- Git
- Visual Studio Code

### Clone the Repository

```bash
git clone https://github.com/sangameshh31/PIZZAPOS.git
cd PIZZAPOS
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file using `.env.example`.

### Start the Application

Commands will be added after the Electron and Express setup is completed.
